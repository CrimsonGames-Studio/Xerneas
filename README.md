# Xerneas

Xerneas is a browser-based experimental AI system built with HTML, CSS, and JavaScript.

The core idea is to generate code dynamically by analyzing and combining structured knowledge stored in JSON files.  
Instead of relying on pre-trained models, Xerneas uses a modular, keyword-based approach to synthesize unique outputs.

---

## 🚀 Features

- 🔍 Keyword-based data navigation
- 🧠 Dynamic code synthesis from JSON snippets
- 📦 Modular architecture (core / modules / utils)
- 📁 Scalable data system (thousands of JSON files supported)
- 🌐 Runs entirely in the browser (GitHub Pages ready)
- 🔌 Plugin system for future extensions

---

## ⚙️ How It Works

1. User provides input via the interface  
2. The system extracts keywords  
3. Relevant JSON snippets are located  
4. Data is filtered and combined  
5. A unique solution is generated  

---

## 📊 Data System

- JSON files are stored in structured directories
- File naming convention:
- snippet00000001.json snippet00000002.json ...
- Each file contains small knowledge fragments (tags + content)
